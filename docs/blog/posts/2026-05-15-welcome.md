# Welcome to the Cube Sandbox Blog


We are launching this blog to share release notes, technical deep-dives, and
community stories around Cube Sandbox — an instant, concurrent, secure and
lightweight sandbox service built for AI agents.

## What you can expect

- **Release notes** for every minor and major version, with migration tips.
- **Architecture posts** about the runtime, networking model (CubeVS), and
  template system.
- **How-to guides** showing real-world integrations with agent frameworks.
- **Community spotlights** featuring use cases from the ecosystem.

Stay tuned — we will be posting regularly.
